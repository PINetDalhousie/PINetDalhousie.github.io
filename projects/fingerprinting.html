Fingerprinting-based Security Services for IoT Network

Fingerprinting, or traffic analysis in general, refers to using traffic metadata emitted by a network device to gain relevant information. Using IoT device fingerprints we can infer activity on those devices, identify their types, and detect any anomalies for security purposes. In this project, PINet aims to:

1) Use programmable network architectures to perform IoT Machine Learning-based traffic analysis on IoT devices and

2) Obfuscate IoT traffic metadata to improve security and privacy of smart home IoT consumers
