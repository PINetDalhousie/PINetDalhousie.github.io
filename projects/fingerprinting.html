Network-accelerated IoT device fingerprinting

Fingerprinting, or traffic analysis in general, refers to using traffic metadata emitted by a network device to gain relevant information. Using IoT device fingerprints, we can infer activity on those devices, identify their types, and detect any anomalies for security purposes. In this project, we use programmable network architectures (e.g. SDN, OpenFlow, P4 switches) to perform fingerprint-based traffic analysis on IoT devices.
